# Basic HTML Website

A multi-page HTML website demonstrating semantic structure, navigation, and SEO optimization - built as a solution to the roadmap.sh [Basic HTML Website](https://roadmap.sh/projects/basic-html-website) project.

## Project Overview

This project demonstrates foundational HTML skills by creating a structured, multi-page website using semantic HTML5. The focus is on clean markup, accessibility, and SEO optimization - with styling to be added in future iterations.

## Requirements Met

- **Multi-Page Structure** - Homepage, Projects, Articles, and Contact pages
- **Semantic HTML Structure** - Proper use of `<header>`, `<main>`, `<section>`, `<article>`, and `<footer>` elements
- **Navigation Bar** - Consistent navigation across all pages linking to all sections
- **SEO Meta Tags** - Description, keywords, and robots meta tags included
- **Contact Form** - Functional contact form with name, email, and message fields
- **Cross-Page Linking** - Internal navigation between all pages

## Technologies Used

- **HTML5** - Semantic markup
- No CSS or JavaScript (minimal approach focusing on structure)

## Local Development

Clone the repository and open any HTML file in your browser:

```bash
git clone https://github.com/aymanbelarbi/basic-html-website.git
cd basic-html-website
open index.html
```

## Website Pages

- **Homepage** - Main landing page with introduction and navigation
- **Projects** - Portfolio of projects and work samples
- **Articles** - Blog posts and written content
- **Contact** - Contact form with fields for name, email, and message

## Learning Outcomes

This project provided hands-on experience with:

- Creating multi-page websites with consistent navigation
- Writing semantic, accessible HTML
